Exporter

Installation

Add this line to your application's Gemfile:

gem 'exporter'

And then execute:

$ bundle

Usage

# Here is the data which you want to export
users = User.all

# export it
Exporter.export(users, :csv).to_file(path_to_file)

Only support export to csv now (more types are coming)
